# training-daily-diary
Simranjeet Singh  
Guru Nanak Dev Engineering College, Ludhiana  
Summer School Training & Internship Programme, IIT Jammu
Course: Data Science & Artificial Intelligence (DS&AI)

Week 1: Data Science & Analytics Foundations (Module 1)

    Day 1: Introduction to Data Science and Data Analytics. Covered basic definitions and the overall data lifecycle.

    Day 2: Studied the analytics workflow and explored various industry applications where data science is utilized.

    Day 3: Learned about different types of data (structured vs. unstructured) and various data collection sources.

    Day 4: Analyzed the crucial role of data in modern business decision-making processes.

    Day 5: Introduction to AI-assisted analytics. Understood how data is processed to generate actionable insights across industries.

Week 2: Python & Excel for Data Analysis (Module 2)

    Day 1: Set up Python for data analysis. Explored core libraries like Pandas and NumPy for data manipulation.

    Day 2: Conducted Exploratory Data Analysis (EDA), focusing on trend analysis, outlier detection, and correlation.

    Day 3: Learned basic data visualization techniques using the Matplotlib library with real-world datasets.

    Day 4: Shifted to advanced Excel for data handling, utilizing formulas, pivot tables, and conditional formatting.

    Day 5: Created interactive dashboards in Excel and utilized AI tools like ChatGPT and Copilot for generating insights and summaries.

Week 3: Machine Learning & Predictive Analytics (Module 3)

    Day 1: Introduction to core Machine Learning concepts, specifically Regression, Classification, and Clustering.

    Day 2: Gained hands-on experience in model building using the scikit-learn library in Python.

    Day 3: Explored various predictive modeling techniques and applied them to sample datasets.

    Day 4: Studied model evaluation techniques and key performance metrics to test model accuracy.

    Day 5: Applied machine learning models to real-world datasets to solidify practical understanding.

Week 4: Data Visualization, AI Automation & Capstone (Module 4)

    Day 1: Focused on advanced data visualization and the art of data storytelling to communicate insights effectively.

    Day 2: Practiced building dynamic dashboards using Power BI and Streamlit.

    Day 3: Analyzed business case studies including sales forecasting, customer segmentation, and social trend analysis.

    Day 4: Learned to automate analytics workflows using Python scripts and various AI APIs.

    Day 5: Began deployment and presentation preparation for the final Capstone Project: an AI-powered analytics dashboard.

Project Title: Audio Mood Classification Using Machine Learning

Description: Developed an end-to-end machine learning pipeline to automatically classify audio tracks into distinct moods (e.g., happy or sad). The project involved using the librosa library to extract audio features from raw music files into a structured dataset, training a Random Forest predictive model, and creating a deployment script to predict the mood of new, unseen audio files. This utilized the data processing, predictive analytics, and machine learning skills acquired during the training module.
